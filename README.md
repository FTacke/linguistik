## Über das Projekt

Dieses Repository enthält die Quelltexte des digitalen Lehrbuchs *Linguistik im Spanischunterricht* (Felix Tacke, Hrsg., 2026),
das an der Philipps-Universität Marburg entstanden ist.  

Das Buch richtet sich an Lehramtsstudierende und Lehrkräfte für das Fach Spanisch. Es verbindet sprachwissenschaftliche Grundlagen mit fachdidaktischer Reflexion und behandelt Fehlerlinguistik, Phonetik und Phonologie, Orthographie, Wortschatz- und Grammatikvermittlung, Variation und Plurizentrik, Sprachwandel sowie den Umgang mit Herkunftssprachen.  

Die Kapitel wurden in Seminaren an der Philipps-Universität Marburg kollaborativ erarbeitet: Studierende haben Textentwürfe verantwortet, die anschließend redaktionell überarbeitet und in eine konsistente Endfassung gebracht wurden.

## Technik

Die Website wird mit [MkDocs](https://www.mkdocs.org/) und dem Theme [Zensical](https://zensical.org/about/) generiert. Das Layout wurde für didaktische Zwecke angepasst (Admonitions, Audio-Elemente, interaktive Karten, mobile Lesbarkeit).
