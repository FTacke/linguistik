# Repo Docs

`repo-docs/` enthaelt interne Repository-Dokumentation. Nichts in diesem Verzeichnis wird vom MkDocs-Build publiziert.

## Struktur

- `audits/` - Repo-Audits und Cleanup-Notizen
- `audio/` - Arbeitsnotizen zum Audio-Workflow
- `design-system/` - interne Layout-, CSS- und Komponenten-Dokumentation
- `editorial/` - redaktionelle Konventionen
- `maps/` - Referenzmaterial fuer Karten und Datenaufbereitung
- `research/` - Analyse-Exports und sonstige Hilfsartefakte

## Regeln

- Oeffentlicher Buchinhalt gehoert nach `docs/`.
- Interne Notizen, Tests und Systemdokumentation gehoeren nach `repo-docs/`.
- Generierte Artefakte bleiben ausserhalb von `docs/` und sollen in der Regel lokal bleiben.
